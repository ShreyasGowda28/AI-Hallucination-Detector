# 🤖 AI Hallucination Detector

An AI-powered hallucination detection system built using **SelfCheckGPT**, **SelfCheckNLI**, and **Ollama (Llama 3.2)**.

This project detects hallucinated information generated by Large Language Models (LLMs) by comparing multiple sampled responses and identifying inconsistent or unsupported statements.

---

## 🚀 Features

- Generate multiple responses using Ollama (Llama 3.2)
- Detect hallucinations using SelfCheckNLI
- Sentence-level hallucination analysis
- Risk classification (LOW / MEDIUM / HIGH)
- Local execution (No OpenAI API required)
- Modular Python architecture

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/ShreyasGowda28/AI-Hallucination-Detector.git

cd AI-Hallucination-Detector
```

### Create Virtual Environment

```bash
python -m venv venv
```

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

Install spaCy model

```bash
python -m spacy download en_core_web_sm
```

Install Ollama

https://ollama.com

Pull Llama 3.2

```bash
ollama pull llama3.2
```


Question

The detector successfully identifies the hallucinated sentence.
